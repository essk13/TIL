# [CSS] class와 id

### [정의]

- **class**
  
  - 그룹으로 묶어 스타일을 지정할 때 사용하는 이름 (.class_name)
  
  - 반복적으로 사용 가능

- **id**
  
  - 한 가지만 지정해서 스타일을 지정할 때 사용하는 이름 (#id_name)
  
  - 하나의 문서에 고유한 id 하나만 사용

### [차이점]

- CSS 적용 우선순위가 다름 (!important > inline > id > class > tag > parent)

- class는 반복 사용 가능하지만 id는 반복 사용 금지
